# 기능 구현 사항

- [x] 카드 추가폼 구현
  - [x] 카드번호를 입력할수 있다.
    - [x] 앞에 8자리는 숫자가 보여지도록, 뒤에 8자리는 가려지도록 구현한다.
  - [x] 만료일을 입력할수 있다.
  - [x] 카드 소유자 이름을 선택적으로 입력할수 있다.
    - [x] 카드 소유자 이름은 최대 30자까지 입력이 가능하다.
  - [x] 보안코드를 입력할수 있다.
    - [x] 보안코드는 3자리수만 가능하다.
    - [x] 옆의 ? 버튼에 커서를 올렸을때 보안코드에 대한 설명을 보여준다.
  - [x] 카드 비밀번호를 입력할수있다.
  - [x] 카드 정보를 적을때마다 카드에 바로 반영된다.
  - [x] 이름을 제외한 모든 입력이 채워질시 다음 버튼이 생긴다.
